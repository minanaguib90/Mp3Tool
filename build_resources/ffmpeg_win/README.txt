Place FFmpeg Windows binaries here before building the Windows .exe.

1. Download a static build that includes ffmpeg.exe and ffprobe.exe, for example:
   https://www.gyan.dev/ffmpeg/builds/
   Get "ffmpeg-release-essentials.zip", extract it, then copy from the bin/ folder:
   - ffmpeg.exe
   - ffprobe.exe

2. Put ffmpeg.exe and ffprobe.exe directly in this folder (build_resources/ffmpeg_win/).

3. Run the Windows build (see BUILD.md in the project root).
